# OFFLINE AI DOCUMENT TRANSLATOR
A Python-based AI application that translates Word, PDF, and Excel documents into multiple languages using the offline Meta NLLB-200 Neural Machine Translation model while preserving document formatting wherever possible.

## FEATURES
* Offline AI-powered translation
* Supports DOCX, PDF and XLSX files
* Multiple source and target language selection
* Preserves document structure and formatting where supported
* User-friendly Streamlit interface
* Download translated documents

## TECHNOLOGIES USED
* Python
* Hugging Face Transformers
* Meta NLLB-200
* PyTorch
* Streamlit
* python-docx
* openpyxl
* PyMuPDF

## INSTALLATION
1. Clone the repository:
   git clone https://github.com/your-username/offline-ai-document-translator.git
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python -m streamlit run app.py

## USAGE
* Upload a DOCX, PDF, or XLSX document
* Select the source and target language
* Click **Translate Document**
* Download the translated document
